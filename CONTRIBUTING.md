# Contributing to DSCI 310 Group 03 Project

We welcome all contributions to this project!

If you notice a bug, or have a feature request,
please open up an issue [here](https://github.com/UBC-DSCI-310-2025W2/dsci-310-group-03/issues).
All contributors must abide by our [code of conduct](CODE_OF_CONDUCT.md). 

Core Members
If you are a core member of the project, please use the GitHub flow workflow:

1. Create a new branch in the main repository for your work
2. Make your changes on that branch
3. Open a pull request from your branch into main
4. A team member will review pull requests within 7 days

Arms-Length Contributors
If you are an external contributor:

1. Fork the repository to your own GitHub account
2. Create a new branch in your fork for your work
3. Make your changes on that branch
4. Open a pull request from your fork's branch into our main branch
5. A team member will review pull requests within 7 days


All contributors must abide by our [code of conduct](CODE_OF_CONDUCT.md).
